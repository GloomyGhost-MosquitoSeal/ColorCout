# ColorCout
Simple cross-platform(?) colored terminal text in C++

## How to use

```cpp
int main() {
    ccout(YELLOW) << "Hello, World!" << cendl;
    ccout(CYAN) << "Hello, World!" << cendl;
    ccout(BLACK) << "Hello, World!" << cendl;
    return 0;
}
```
